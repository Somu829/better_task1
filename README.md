# Task Management API - Comments Management

This project provides a backend API for managing comments associated with tasks. The API allows users to create, retrieve, update, and delete comments on tasks, using standard CRUD operations.

## Features

- **Add a Comment**: Allows users to add a comment to a task.
- **List Comments**: Retrieve all comments associated with a specific task.
- **Edit a Comment**: Edit the content of an existing comment.
- **Delete a Comment**: Delete a specific comment from a task.
